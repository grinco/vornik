# Vornik

**Vornik** is an open-source multi-agent orchestration fabric: it runs a swarm of
LLM agents as durable, leased tasks — with workflows, tool access over MCP, and a
control CLI — on a single deployable daemon.

> **Editions.** This repository is **Vornik Community Edition** (AGPL-3.0). A
> proprietary **Enterprise Edition** adds advanced capabilities on top of the same
> core — see [editions.md](editions.md) for the feature matrix. Community is fully
> usable on its own for personal and small-team use.
