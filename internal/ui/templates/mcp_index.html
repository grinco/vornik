How it works. The daemon connects to every server listed in its top-level mcp.servers block at startup, fetches the tool catalog, and caches it for 5 minutes. Stale entries refresh in the background — the page never waits for a slow server.

Auto-discovery ≠ auto-grant. Listing a server here does not give any project access to its tools. Granting a server to a project still requires adding it to that project's own mcp.servers list.
